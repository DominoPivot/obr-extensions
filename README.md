# DominoPivot's extensions for Owlbear Rodeo

This repository will hold the published version of extensions I make for [Owlbear Rodeo](https://www.owlbear.rodeo/), a browser-based virtual tabletop software.
The repository exists primarily to allow all my extensions to be served through Github Pages from one custom subdomain, [obrx.dominopivot.com](https://obrx.dominopivot.com/).
